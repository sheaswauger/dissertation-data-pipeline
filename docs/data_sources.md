# Data Sources

Master reference for all dissertation data sources.

## Primary Data

### Implicit Association Test (IAT)

- **Source:** Project Implicit via OSF
- **Race IAT:** https://osf.io/gwofk/
- **Gender-Career IAT:** https://osf.io/abxq7/
- **Years:** 2010-2024
- **Access:** Public, no registration required

### State Legislation

- **Source:** LegiScan API
- **URL:** https://legiscan.com/
- **Years:** 2010-2025
- **Access:** API key required (free tier available)
